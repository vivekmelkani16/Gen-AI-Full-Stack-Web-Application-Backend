# 🤖 AI-Powered Full Stack Web Application (Backend)

A backend system that integrates Generative AI (Google Gemini API) to provide intelligent responses based on user input, resume data, and job descriptions.

---

## 🚀 Features

- AI-powered response generation using Google Gemini API
- Resume analysis and interview report generation
- RESTful API architecture
- Secure backend with JWT authentication
- File handling and API integration

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- MongoDB
- Google Gemini API
- JWT Authentication

---

## 📡 API Endpoints

### AI Features
- POST /api/ai/generate-report
- POST /api/ai/analyze-resume

### Auth
- POST /api/auth/register
- POST /api/auth/login
